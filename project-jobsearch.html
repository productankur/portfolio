User Journey

How It Works

01
Search for Jobs
Enter a keyword and location in the sidebar. The app calls the USAJobs API and returns live federal job listings instantly.
fetch_usajobs(keyword, location, results_per_page)
02
Browse Listings
Jobs appear as expandable cards showing title, agency, location, summary, and an Apply link. Each card has a Generate button.
st.expander() + st.session_state["jobs"]
03
Agents Activate
All 3 CrewAI agents run sequentially with delays to respect Groq rate limits. A spinner keeps the UI responsive via async execution.
crew.kickoff_async() via uvloop
04
Materials Generated
JD analysis, tailored cover letter, and LinkedIn message are displayed inline and saved automatically to the data/ directory.
output_file="data/cover_letters/cover_letter.md"
05
Apply with Confidence
Use the generated materials to apply with a matched resume summary, personalized cover letter, and a LinkedIn message ready to send.
log_application() + save_cover_letter_file()
